# Licencia y Descargo de Responsabilidad

Esta sección contiene información legal importante sobre el uso, distribución y responsabilidades asociadas con el proyecto WSPR Beakon.

## Licencia

Este proyecto está licenciado bajo la **Licencia de Hardware Abierto CERN Versión 2 - Fuertemente Recíproca (CERN-OHL-S v2)**.

### Lo Que Esta Licencia Permite

- **Usar** el diseño de hardware para cualquier propósito
- **Estudiar** cómo funciona el hardware y aprender del diseño
- **Modificar** el diseño para adaptarlo a sus necesidades específicas
- **Compartir** copias del diseño original o modificado con otros

### Condición Fuertemente Recíproca

La condición "Fuertemente Recíproca" significa que si distribuye productos basados en este diseño, **también debe compartir cualquier modificación bajo la misma licencia**, asegurando que el diseño de hardware permanezca abierto y accesible para la comunidad.

### Texto Completo de la Licencia

Para los términos completos de la licencia, consulte:
- **CERN-OHL-S v2**: [https://ohwr.org/cern_ohl_s_v2.txt](https://ohwr.org/cern_ohl_s_v2.txt)
- **Repositorio del Proyecto**: [https://github.com/danielcharrua/wspr-beakon](https://github.com/danielcharrua/wspr-beakon)

## Descargo de Responsabilidad

### Descargo General

Este proyecto se comparte **tal como está, sin garantía de ningún tipo**. Los autores y contribuyentes no hacen representaciones o garantías sobre la precisión, completitud o confiabilidad de la información, diseños o software proporcionados.

### Responsabilidad del Usuario

**Cada persona es responsable de:**
- Ensamblar y usar su propio sistema WSPR Beakon
- Asegurar el cumplimiento con las regulaciones y leyes de radio locales
- Obtener la licencia de radioaficionado apropiada donde sea requerida
- Usar medidas de filtrado RF y seguridad apropiadas
- Probar y verificar el rendimiento de su ensamblaje

### Cumplimiento Regulatorio

:::danger Requisitos Regulatorios
**DEBE cumplir con las regulaciones de radio locales** antes de operar cualquier sistema WSPR Beakon. Esto incluye pero no se limita a:

- **Licencia de Radioaficionado**: Se requiere licencia válida para transmisión
- **Limitaciones de Potencia**: Respetar límites de potencia locales para cada banda
- **Emisiones Espurias**: Uso obligatorio de filtrado apropiado
- **Planes de Banda**: Operar solo dentro de frecuencias WSPR designadas
- **Identificación**: Identificación adecuada de estación según requiera la ley
:::

### Seguridad RF y Filtrado

:::warning Filtrado RF Obligatorio
**EL USO DE FILTRADO RF APROPIADO ES ABSOLUTAMENTE OBLIGATORIO.** El módulo Si5351 genera armónicos que deben filtrarse antes de conectar a cualquier sistema de antena. El no usar filtrado apropiado puede:

- Causar interferencia a otros servicios de radio
- Violar requisitos regulatorios
- Resultar en penalidades legales
- Dañar otros equipos
:::

### Seguridad de Fuente de Alimentación

:::warning Seguridad Eléctrica
- Use fuentes de alimentación apropiadas clasificadas para los requisitos del circuito
- Desconecte la alimentación externa antes de conectar cables de programación
- Siga prácticas de seguridad eléctrica apropiadas durante ensamblaje y operación
- Tenga en cuenta que conexiones de alimentación inadecuadas pueden dañar equipos o computadoras
:::

**Este proyecto se proporciona sin garantía, incluyendo pero no limitado a:**

- Ha leído y entendido este descargo de responsabilidad
